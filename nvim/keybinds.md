## Navigation

- **Cursor Movements**:
  - `h`: Move cursor left
  - `j`: Move cursor down
  - `k`: Move cursor up
  - `l`: Move cursor right

## Editing

- **Insert Mode**:
  - `i`: Insert before the cursor
  - `I`: Insert at the beginning of the line
  ...

## Visual Mode

- **Basic Selections**:
  - `v`: Start visual selection
  - `V`: Select entire lines
  ...

## Searching and Replacing

- **Searching**:
  - `/`: Search forward
  - `?`: Search backward
  ...
  
## Buffers and Tabs

- **Buffer Operations**:
  - `:bnext`: Go to next buffer
  - `:bprev`: Go to previous buffer
  ...

## Windows and Splits

- **Window Management**:
  - `:split`: Split window horizontally
  - `:vsplit`: Split window vertically
  ...

## File Operations

- **File Commands**:
  - `:e`: Open file
  - `:w`: Save file
  ...
  
## Command Line Mode

- **Command-Line Operations**:
  - `:`: Enter command mode
  - `/`: Search
  ...

## Macros and Registers

- **Macro Operations**:
  - `q`: Start recording macro
  - `@`: Play macro
  ...

## Miscellaneous

- **Others**:
  - `u`: Undo
  - `.`: Repeat last command
  ...
